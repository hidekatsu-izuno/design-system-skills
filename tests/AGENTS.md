# tests ディレクトリについて

このディレクトリは、`tailwindcss` を用いた UI モックのサンプルを配置するための領域です。

各サンプル用フォルダには、以下のいずれか、または両方を配置します。

- UI モック実装一式
- モックに対応する画面設計書や補足ドキュメント

現在の想定:

- `sample1`: マスタ管理システムのダッシュボード UI モック用サンプル

作成方針:

- 見た目確認を主目的とした軽量なサンプルとする
- `tailwindcss` ベースで、画面構造と情報優先度が分かる実装を優先する
- 必要に応じて各サンプル配下の `AGENTS.md` で個別要件を定義する
